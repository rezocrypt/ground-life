# Getting Started

To get a local copy up and running follow these simple example steps.

## Prerequisites

You need to have the following things installed on your pc.

- Install any browser. 
- Install `git`.

## Installation

Here are steps to download, build and install the following software.

1. Clone the repo
```sh 
git clone https://github.com/rezocrypt/ground-life
```

2. Go to the directory
```sh
cd ground-life 
```

3. Open the index.html file in any browser.
